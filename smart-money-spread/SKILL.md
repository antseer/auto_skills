# Smart Money 价差

## 功能说明

计算机构买家（鲸鱼钱包、ETF 资金流、企业国库购入）与散户情绪（社交情绪、恐惧贪婪指数、资金费率）之间的实时背离价差。产出 -100 到 +100 的量化分数、体制标签、置信等级（高/中/低）、多层因果链解读、失效条件、以及针对不同认知水平的分层解读。当两者分歧时（机构积累 + 散户恐慌，或机构分销 + 散户贪婪）价值最大。

## 适用场景

用户想知道机构买家和散户情绪是否朝同一方向移动。适用于: 重大机构 BTC 购入被披露; 散户情绪急剧下降; 即将调整加密或 BTC 关联股票仓位; 恐慌标题出现时做理性检查。

**触发短语**: "鲸鱼在买还是在卖", "smart money 在这次下跌中积累吗", "机构 vs 散户价差是多少", "该不该担心这次暴跌", "Saylor 刚买了——散户怎么看", "入场前查一下 smart money spread", "恐惧有根据还是机构在买", "smart money vs scared money"

## 输入参数

| 参数 | 必须 | 默认值 | 说明 |
|------|------|--------|------|
| 资产 | 否 | BTC | 要分析的资产 |
| 时间窗口 | 否 | 7d | 回溯期 |

## 执行步骤

### 第一步：收集机构信念信号

```
ant_smart_money(query_type="whale_movements", asset="BTC", timeframe="7d")
```
记录: whale_net_flow, whale_buy_count, whale_total_count
→ whale_conviction_ratio = whale_buy_count / whale_total_count

```
ant_fund_flow(query_type="etf_flows", asset="BTC", timeframe="7d")
```
记录: etf_net_flow_usd, etf_flow_trend

```
ant_macro_economics(query_type="institutional_activity", asset="BTC")
```
记录: treasury_purchases (实体、BTC量、USD量、成本基础), treasury_vs_market_price

计算 **institutional_conviction_score** (0-100):
```
whale_component = whale_conviction_ratio * 40          # 最高 40 分
etf_component = normalize(etf_net_flow_usd, -500M..+500M) * 30   # 最高 30 分
treasury_component = (30 if 7天内有折价购入 else 15 if 溢价购入 else 0)  # 最高 30 分
institutional_conviction_score = round(whale_component + etf_component + treasury_component)
```

### 第二步：收集散户恐惧信号

```
ant_market_sentiment(query_type="topics_list")
```
筛选加密相关话题，前 5 个:
```
ant_market_sentiment(query_type="topic_posts", topic="<topic>")
```
计算: social_sentiment_mean (-1..+1), social_bearish_pct

```
ant_market_indicators(query_type="fear_greed_index")
```
记录: fear_greed_score (0-100), fear_greed_label

```
ant_futures_market_structure(query_type="funding_rates", asset="BTC", exchanges=["binance","bybit","dydx"])
```
记录: avg_funding_rate, funding_rate_percentile (vs 90天)

计算 **retail_fear_score** (0-100, 100=最大恐惧):
```
sentiment_component = (1 - (social_sentiment_mean + 1) / 2) * 30  # 最高 30
fear_greed_component = (100 - fear_greed_score) * 0.4             # 最高 40
funding_component = normalize(-avg_funding_rate, -0.05..+0.05) * 30  # 最高 30
retail_fear_score = round(sentiment_component + fear_greed_component + funding_component)
```

### 第二.五步：多层因果链推导

**此步为深度分析核心——不可跳过。** 对每个信号层级，推导至少 2 层 "为什么"，形成完整因果链。

**机构信念因果链（必须推导）**：

1. **鲸鱼行为因果链**:
   - 观察: whale_conviction_ratio = X（Y/Z 钱包净买入）
   - 第一层为什么: 检查鲸鱼买入的价格区间 vs 当前价格 → 是在折价区间积累？还是追涨？
   - 第二层为什么: 如果折价积累 → 推断鲸鱼认为当前价格低于其内在估值基准 → 检查这些钱包的历史买入模式（是否在过去的底部区间也有类似行为）
   - 结论: "[鲸鱼积累/分销]的因果链: [观察] → 因为[第一层] → 意味着[第二层]"

2. **ETF 资金流因果链**:
   - 观察: etf_net_flow = $X（趋势: 加速/减速/反转）
   - 第一层为什么: ETF 资金流的趋势方向比绝对值更重要 → 加速流入意味着新的机构配置决策正在执行，而非存量调仓
   - 第二层为什么: 检查 ETF 流入是否与宏观利率预期变化同步 → 如果利率预期下降 + ETF 流入加速，说明机构在利率周期拐点前预先配置；如果利率预期上升但 ETF 仍流入，说明机构有独立于利率的配置逻辑（如通胀对冲）
   - 结论: "[ETF 资金流]的因果链: [观察] → 因为[第一层] → 意味着[第二层]"

3. **国库购入因果链**:
   - 观察: [实体] 以 $X 购入 Y BTC，vs 其平均成本基础 $Z
   - 第一层为什么: 购入价格 vs 成本基础的关系 → 低于成本基础=买弱势（高信念度），高于成本基础=买强势（追涨风险存在）
   - 第二层为什么: 检查购入规模 vs 该实体总持仓占比 → 如果此次购入占总持仓 >5%，这是战略级配置变更；如果 <1%，可能只是常规 DCA
   - 结论: "[国库购入]的因果链: [观察] → 因为[第一层] → 意味着[第二层]"

**散户恐惧因果链（必须推导）**：

1. **社交情绪因果链**:
   - 观察: social_sentiment_mean = X, bearish_pct = Y%
   - 第一层为什么: 判断看空情绪是由特定事件驱动（如黑客攻击、监管新闻）还是弥漫性恐慌（无明确催化剂）
   - 第二层为什么: 事件驱动的恐惧通常在 48-72h 内消退（历史上事件冲击后情绪回归均值的中位时间）；弥漫性恐慌可能持续数周，暗示市场结构问题
   - 结论: "[社交恐惧]的因果链: [观察] → 因为[第一层] → 意味着[第二层]"

2. **资金费率因果链**:
   - 观察: avg_funding_rate = X, percentile = Y vs 90天
   - 第一层为什么: 负费率 → 空头主导 → 但空头拥挤本身是潜在的逼空燃料
   - 第二层为什么: 检查费率负值持续天数 → 如果持续 >3天且加深 → 不是短期技术性超卖，而是市场参与者结构性转空；如果仅持续 1-2天 → 更可能是杠杆清洗后的短暂超调
   - 结论: "[资金费率]的因果链: [观察] → 因为[第一层] → 意味着[第二层]"

### 第三步：计算背离价差

```
raw_spread = institutional_conviction_score - (100 - retail_fear_score)
normalized_spread = round(raw_spread)  # 范围约 -100 到 +100
```

体制标签:

| 价差 | 体制 | 解读 |
|------|------|------|
| +60 到 +100 | 强机构积累 | 机构大举买入 + 散户恐慌。历史上利于中期多头。 |
| +30 到 +59 | 机构积累 | 机构净买入，散户恐惧。背离明显但不极端。 |
| -29 到 +29 | 中性/一致 | 双方大致方向一致。无背离信号。 |
| -59 到 -30 | 机构分销 | 机构净卖出 + 散户自满/贪婪。需谨慎。 |
| -100 到 -60 | 强机构分销 | 机构大举分销 + 散户追涨。历史上先于回调。 |

### 第三.五步：跨源交叉验证与置信等级

**此步决定输出的置信等级——不可跳过。** 单一数据源的结论不可获得"高置信"评级。

**验证规则**:

将信号分为两个独立验证组:

- **机构信号组**: 鲸鱼钱包行为、ETF 资金流、国库购入（3 个信号）
- **散户信号组**: 社交情绪、恐惧贪婪指数、资金费率（3 个信号）

对每组分别计算方向一致性:
```
机构信号一致性 = (方向一致的信号数) / (总信号数)
散户信号一致性 = (方向一致的信号数) / (总信号数)
```

例如: 鲸鱼净买入(看多) + ETF 流入(看多) + 国库折价购入(看多) = 3/3 一致 = 100%
例如: 鲸鱼净买入(看多) + ETF 流出(看空) + 无国库购入(中性) = 1/3 方向明确看多 = 33%

**置信等级判定**:

| 条件 | 置信等级 | 输出标注 |
|------|---------|---------|
| 两组一致性均 ≥ 67% 且方向相反 | **高置信** | "高置信: 机构信号和散户信号内部一致，且方向相反——背离信号可靠" |
| 一组一致性 ≥ 67%，另一组 < 67% | **中置信** | "中置信: [X组]信号一致但[Y组]信号混杂——背离可能存在但有噪音" |
| 两组一致性均 < 67% | **低置信** | "低置信: 机构和散户信号均不一致——当前无可靠背离判断" |

**跨组交叉验证异常检测**:

如果出现以下矛盾信号，必须在输出中醒目标注:
- 鲸鱼净买入但 ETF 净流出 → "注意: 链上鲸鱼和机构 ETF 投资者行为相反，可能表明不同类型机构有分歧"
- 恐惧贪婪指数恐惧但资金费率正值（多头拥挤）→ "注意: 散户嘴上说怕但持仓没减，恐惧可能是口头恐惧而非实际去杠杆"
- 国库大额折价购入但鲸鱼净卖出 → "注意: 上市公司在买但链上大户在卖，可能是信息不对称或时间差"

### 第四步：宏观叙事交叉检查

**目的**: 检测宏观环境是否与价差体制矛盾，避免在宏观/加密叙事强分歧期给出误导性信号。

```
ant_macro_economics(query_type="interest_rate_outlook")
```
记录: rate_direction (鸽派/鹰派/中性), dollar_trend (走强/走弱/中性)

```
ant_macro_economics(query_type="risk_appetite_indicators")
```
记录: risk_on_off_signal (风险偏好/风险规避/中性)

**宏观一致性检查**:

| 价差体制 | 宏观信号 | 判定 |
|---------|---------|------|
| 强机构积累 (+60~+100) | 利率鸽派 + 美元走弱 | 一致: 宏观支持机构积累逻辑 |
| 强机构积累 (+60~+100) | 利率鹰派 + 美元走强 | **矛盾**: 宏观逆风可能让机构积累失效 |
| 机构分销 (-30~-59) | 利率鹰派 + 风险规避 | 一致: 宏观环境恶化印证机构分销 |
| 机构分销 (-30~-59) | 利率鸽派 + 风险偏好 | **矛盾**: 宏观利好可能让机构分销只是短期获利了结 |

如果宏观信号与价差体制矛盾:
- 将置信等级降一级（高→中，中→低）
- 在输出中增加 "宏观叙事冲突" 段落:
  "当前价差体制为[X]，但宏观环境显示[Y]。历史上当机构行为与宏观趋势矛盾时，宏观趋势在 [60%/70%] 的情况下最终主导中期走势。建议缩小仓位或缩短持有周期。"

### 第五步：生成分层解读

**用户类型判断规则**（从对话上下文推断）：

| 用户类型 | 判断条件 |
|----------|---------|
| 通俗用户 | 持仓 < $5,000，或自述为新手，或提问中未使用专业术语 |
| 分析用户 | 持仓 ≥ $5,000，或主动使用"资金费率"/"IV"/"8-K"等术语，或要求详细数据 |

**通俗解读写作规则（严格执行）**：

通俗解读必须满足以下所有条件：
1. **严禁使用**以下术语（无论何种用户类型均在通俗解读中禁止）：资金费率、funding rate、ETF 资金流、whale、鲸鱼净流向、机构信念分数、散户恐惧分数、IV、期权、价差、normalization、分位数、percentile、8-K、基础差
2. **必须使用**具体数字代替术语：不写"鲸鱼净买入"，写"X 个大 BTC 持有者中有 Y 个本周加仓"
3. **必须给出一句行动方向提示**（不是买卖建议，而是情绪定性）：例如"标题在说恐慌，但大买家在加仓" 或 "大持有者本周在卖出，散户仍在追涨"
4. **必须包含一句置信度说明**: 例如"这个判断基于多个来源数据一致指向同一方向，可靠度较高" 或 "这个判断的可靠度一般，因为部分数据信号互相矛盾"
5. **必须包含一句失效提醒**: 例如"如果明天出现 XX 情况，今天的判断可能翻转"

通俗用户额外输出（在完整报告前方插入）：
```
通俗解读: "[X] 个大 BTC 持有者中有 [Y] 个本周加仓。
市场情绪评级: [极度恐惧 | 恐惧 | 中性 | 贪婪 | 极度贪婪]（[fear_greed_score] 分）。
这意味着: [一句话解释当前背离对普通持有者的实际含义，不含任何专业术语]。
可靠度: [高 | 中 | 低] — [一句话解释为什么]。
注意: 如果 [一句话说明失效条件]，这个判断会失效。"
```

所有用户都输出完整分析版（见输出格式）。

## 使用示例

**用户**: "Saylor 又宣布 BTC 购入了。该担心暴跌还是机构在买低？"

→ 执行全部步骤。

**机构信念链**:
- 鲸鱼: 8/11 钱包净买入 → 买入均价低于当前价 3.2% → 与 2024 年 9 月底部积累模式相似，暗示鲸鱼判断当前为折价区间
- ETF: +$412M 加速流入(上周 +$280M) → 流入加速而非减速 → 与利率预期下降同步，说明机构在利率周期拐点前预先配置
- 国库: Strategy 以低于成本基础 10.5% 购入 → 买弱势 → 此次购入占 Strategy 总持仓 2.8%，属于战略级增持而非日常 DCA

**散户恐惧链**:
- 情绪: 社交看空 62% → 恐慌由"暴跌"标题驱动而非结构性事件 → 事件驱动恐惧历史上 48-72h 内消退
- 费率: -0.025% (第 8 百分位) → 空头拥挤 → 仅持续 1 天，更可能是杠杆清洗后超调而非结构性转空

**交叉验证**: 机构信号 3/3 一致(看多)，散户信号 3/3 一致(恐惧) → 高置信
**宏观检查**: 利率鸽派 + 美元走弱 → 与机构积累一致

价差 +68 = 强机构积累，置信: 高

**通俗解读**: "11 个大 BTC 持有者中有 8 个本周加仓。标题说恐慌，但最大买家把这当作买入机会。多个数据来源同时指向同一方向，这个判断可靠度较高。小额组合不必恐慌卖出。注意: 如果未来 3 天大买家开始卖出或美联储意外转鹰，今天的判断可能翻转。"

## 反模式

- **本技能不提供买卖建议。** 只计算背离价差和描述机构 vs 散户行为。
- **本技能不预测价格方向。** 高机构信念分数不保证反弹。
- **本技能不追踪分析师观点。** 只追踪机构实际资金行为（钱包流、ETF 购入、8-K 文件）。
- **本技能不要求用户懂资金费率或 8-K 文件。** 通俗解读部分翻译一切。
- **本技能不替代仓位大小工具。** 提供背离信号辅助决策，不计算仓位/止损。
- **本技能不覆盖无机构存在的资产。** 低市值山寨币/meme 币缺乏机构数据时明确说明无法计算。
- **本技能不把过期数据当新鲜数据。** 数据新鲜度部分展示每个输入的年龄。鲸鱼数据 > 4 小时或资金费率 > 15 分钟时标注置信度降低。
- **本技能不输出无失效条件的判断。** 每次输出的体制判断必须附带至少 2 条失效条件。
- **本技能不在信号矛盾时给出高置信判断。** 当机构信号组或散户信号组内部方向不一致时，自动降低置信等级。
- **本技能不忽视宏观逆风。** 当宏观叙事与价差体制矛盾时，强制标注冲突并降低置信。
