# Smart Money Spread - 需求文档 (PRD)

## 需求概述

当用户想知道机构买家（鲸鱼钱包、企业国库、ETF 资金流）和散户情绪是否朝同一方向移动时，需要一个工具计算实时的"机构信念 vs 散户恐惧"价差并返回量化背离分数。目前这些信号分散在 Arkham、CoinGlass、alternative.me、Edgar 等多个平台，没有工具将它们统一为单一指标。这是 2026-04-07 讨论中全部 4/4 persona 独立引用的最高优先级新洞察。

## 用户画像

### Alice -- DeFi 全栈独立开发者 / 链上分析师

- **角色**: 每日使用 Claude Code 写 Solidity 合约、分析 gas、构建链上数据管道
- **工具栈**: Solidity, TypeScript, Python, Hardhat, Dune Analytics, The Graph, Arkham, CoinGlass, Claude Code（每日）
- **痛点**: Alice 已经追踪鲸鱼钱包积累作为日常工作流的一部分，但从未将机构活动与散户恐慌之间的差距形式化为单一量化信号。3 月 28 日降息恐慌期间，她花了 2 小时手动在 Arkham、CoinGlass 和 Twitter 之间编撰分析帖子。一个计算好的价差本可在几分钟内给她结果。

### Bob -- 散户交易者 / 业余量化

- **角色**: 追踪 6-8 份宏观新闻简报，管理 20-30 只股票和小额加密配置
- **工具栈**: Python, Pine Script, TradingView, Google Sheets, Substack, Claude Code（回测脚本）
- **痛点**: Bob 的机构购入数据通过回复帖子延迟 2-3 天到达，而散户情绪立即出现在他面前。Strategy 宣布以 $67,718 购入 $330M BTC 时，Bob 直到两天后才注意到。到他手动拼凑全貌时，背离窗口已关闭，交易已重新定价。

### Carol -- 全栈 SaaS 开发者 / 跨资产交易者

- **角色**: 交易 MSTR、MARA、BTC 现货和预测市场
- **工具栈**: Next.js, Supabase, Tailwind, TypeScript, Binance, CoinGlass, Polymarket, Edgar, Claude Code（每日）
- **痛点**: Carol 拥有价差的所有原始数据输入（Saylor 购入、资金费率、预测市场赔率）但从未将"机构行动 vs 散户恐惧"形式化为独立指标。两周前她发现 8-K 显示购入 3,000 BTC、恐惧贪婪指数 28、MSTR NAV 折价 8%——在她手动拼凑全貌时股票已从低点反弹 3%。

### Dave -- 加密新手 / 学习型用户

- **角色**: 持有约 $400 BTC（Cash App）和少量股票
- **工具栈**: Python 基础, Jupyter Notebook, Cash App, Reddit, Claude Code（学习用）
- **痛点**: Dave 无法阅读 Etherscan、解析 8-K 文件或解读资金费率。当 CNBC 推送恐慌标题时，他差点恐慌卖出，花 45 分钟在 Reddit 寻找矛盾答案。他需要一个通俗信号："10 个大买家中有 7 个在积累，散户恐惧度 85%"——10 秒内在午休时可以理解。

## 用户故事

1. **作为 Alice**，当宏观新闻发布且资金费率与鲸鱼钱包行为出现尖锐分歧时，我想获得单一计算好的背离分数，以便在信号窗口关闭前为 DAO 和 Telegram 受众发布及时的链上分析帖子。

2. **作为 Bob**，当机构 BTC 购入通过 8-K 文件或鲸鱼警报披露时，我想立即看到它与当前散户情绪的对比分数，以便在 2-3 天背离窗口内决定是否增加 BTC 配置。

3. **作为 Carol**，当我即将进入 BTC 国库股仓位时，我想获得结合企业国库积累、散户恐惧指数和 NAV 溢价数据的实时价差，以便将入场时机对准背离。

4. **作为 Dave**，当恐慌标题出现且我想知道是否该担心我的小额 BTC 仓位时，我想获得一个通俗信号告诉我大买家是在积累还是卖出，以便 30 秒内做出冷静决策。

## 需求边界

### 做什么
- 计算机构信念分数（鲸鱼积累 + ETF 资金流 + 国库购入）
- 计算散户恐惧分数（社交情绪 + 恐惧贪婪指数 + 资金费率）
- 产出量化背离价差（-100 到 +100）
- 提供体制标签和通俗解读
- 展示数据新鲜度

### 不做什么
- 不提供买入/卖出建议
- 不预测价格方向
- 不追踪分析师观点（只追踪机构资金行为）
- 不覆盖没有机构存在的低市值山寨币
- 不替代仓位大小工具
- 不将过期数据当作新鲜数据展示
