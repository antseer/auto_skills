```

  ETF AUM 7d 变化: $87.87B -> $90.97B = +$3.1B
  etf_component = normalize(+3100M, -500M..+500M) = 1.0 (超出上限，cap at max) * 30 = 30
  注: ETF 净流入远超 $500M 基线，组件满分

国库组件:
  treasury_component = 0 (窗口内无公开 8-K 折价购入)

基础分: round(22.8 + 30 + 0) = 53

定性调整 (+15):
  + Coinbase 溢价从 -$2.77 翻转至 +$13.64 (强正，暗示美国机构买入)
  + 鲸鱼 top10 大额提币占比高 (4/8: top10 占流出 30.5%)

```

### 散户恐惧分数 (retail_fear_score)

```
社交情绪组件:
  social_sentiment = 70/100 -> 归一化至 -1..+1: (70-50)/50 = +0.4
  sentiment_component = (1 - (0.4 + 1) / 2) * 30 = (1 - 0.7) * 30 = 9.0

恐惧贪婪组件:
  CGDI 1585.8, 归一化至 0-100 scale 约 50 (中位区间)
  fear_greed_component = (100 - 50) * 0.4 = 20.0

资金费率组件:
  avg_funding_rate = +0.0064 (Binance) / +0.0035 (OI加权)
  使用 OI 加权: funding_component = normalize(-0.0035, -0.05..+0.05) * 30
  = ((0.05 - 0.0035) / 0.10) * 30 = 0.465 * 30 = 13.95

基础分: round(9.0 + 20.0 + 13.95) = 43

定性调整 (+6):
  + social_dominance 飙升 2.4x (29% -> 71%) 显示焦虑集中
  + sentiment 从 73 降至 70，相对 7d +6.5% 涨幅严重滞后 (隐性恐惧)
  + galaxy_score 59.6 < sentiment 70 (热议但不信)

retail_fear_score = 49
```

### 背离价差

```
raw_spread = institutional_conviction_score - (100 - retail_fear_score)
           = 68 - (100 - 49)
           = 68 - 51
           = +17

定性调整 (+21):
  鲸鱼大额提币模式 + ETF AUM 持续增长 + Coinbase 溢价强正 -> 机构信号一致性高
  vs 口头恐惧但量化不极端 -> 散户信号噪音大
  综合判断背离程度超出纯公式计算

最终价差 = +38 (机构积累区间 +30 ~ +59)
```
